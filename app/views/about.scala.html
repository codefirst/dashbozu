Overview

Dashbozu is a realtime and croscutting view for some other services(e.g. Jenkins, git, Redmine).

Features

  • Integrate some other serviecs
  • Realtime notification(using Pusher)
  • iPhone UI
  • Push notification for iPhone/iPad(usnig Boxcar)
